# 💧 스마트 미터 온디바이스(On-device) 누수 감지 시스템
> **시계열 예측 모델 최적화 및 삼중 검증 기반 실시간 진단 파이프라인 구축**

## 🎯 프로젝트 개요
상수도 스마트 미터 데이터를 활용하여 저사양 현장 기기(On-device)에서도 구동 가능한 **고정밀 누수 진단 AI 시스템**을 개발했습니다. 데이터 감사(Audit)를 통한 계통적 오류 복원과 **삼중 검증(Triple-Check)** 로직을 통해 실무 투입이 가능한 수준의 알람 신뢰도를 확보했습니다.

## 🚀 핵심 성과 (Key Results)
1. **데이터 무결성 확보:** 전수 조사를 통해 발견된 **유량 음수 데이터(35%)를 절대값 처리(abs)로 복원**하여 학습 데이터의 손실을 방지하고 예측력 강화
2. **통합 전처리 파이프라인:** EDA부터 클리핑, **30분 단위 리샘플링**, 스케일링을 단일 공정으로 표준화하여 모델 간 비교 실험의 객관성 확보
3. **삼중 검증(Triple-Check) 시스템:** 통계적 기법(**Z-score**), 기계학습(**Isolation Forest**), 실무 로직(**지속성 필터**)을 결합하여 오탐지 최소화
4. **모델 전략 최적화:** 딥러닝(TSMixer)의 평활화 문제를 **LightGBM과 특성 공학(Lag Features)** 조합으로 해결하여 실시간 변동성 추종 성능 극대화

## 📊 모델 성능 평가 (Performance Metrics)
### 1. 예측 성능 (Regression)
- **수압 예측 모델 (Pressure):** **R² Score 0.86** / **MAE 0.05** (매우 우수)
- **유량 예측 모델 (Flow):** **R² Score 0.65** / **MAE 0.11** (실무 적합 수준)
*스케일링 지표뿐만 아니라 실측 단위 환산을 통한 현장 오차 범위(MAE) 검증 완료*

### 2. 실시간 진단 신뢰도 (Diagnostic Reliability)
- **진단 신뢰도 (F1-Score):** **0.76** (전처리 고도화 전 0.56 대비 **35% 향상**)
- **진단 정밀도 (Precision):** **75%** (알람 발생 시 실제 이상 상황일 확률)
- **진단 재현율 (Recall):** **77%** (실제 누수 징후 중 모델이 포착한 비율)
